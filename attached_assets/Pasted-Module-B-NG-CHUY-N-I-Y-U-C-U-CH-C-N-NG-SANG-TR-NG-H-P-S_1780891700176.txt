Lưu trữ tập trung		API Upload (File Service)	Hệ thống 		Hỗ trợ upload đa phương thức; Kiểm soát định dạng; Hỗ trợ Multipart Upload cho các file dung lượng lớn để đảm bảo tính ổn định				
		API View truy xuất	Hệ thống 		Cung cấp Endpoint trả về luồng dữ liệu (Stream) với Header Content-Type phù hợp (ví dụ: image/jpeg) để hiển thị trực tiếp trên thẻ <img> của trình duyệt				
		API Hỗ trợ tham số tùy chỉnh kích thước ảnh qua URL	Hệ thống 		Hỗ trợ tùy chỉnh kích thước ảnh upload lên hệ thống cho phù hợp, ví dụ:?width=200&height=200
